# diabetes_streamlit_prediction_-22
🩺 Diabetes Prediction App

A simple Diabetes Prediction system built using Python Machine Learning and Streamlit.
The project uses Logistic Regression (simple script) and Random Forest (Streamlit app) to predict whether a person is diabetic based on medical inputs.

✨ Features

Predicts Diabetic / Non-Diabetic

Machine Learning based prediction

Simple Logistic Regression model (Python)

Interactive Streamlit web app (Random Forest)

User-friendly blue & white UI

Displays prediction result with probability

▶ How to Run the Project
Step 1: Install Required Libraries

Open terminal and run:

pip install pandas numpy scikit-learn streamlit
Step 2: Run Logistic Regression Model
python logistic_regression.py

This will train the model and show prediction output in terminal.

Step 3: Run Streamlit App
streamlit run app.py
Step 4: Open in Browser

After running Streamlit, open the link shown in terminal (usually):

http://localhost:8501

Enter the medical values and click Predict.
